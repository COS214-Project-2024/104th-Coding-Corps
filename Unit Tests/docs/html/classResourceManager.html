Add a resource to the ResourceManager.

This method adds a specified quantity of a resource to the ResourceManager. If the resource already exists, its quantity is increased. If it does not exist, it creates a new resource using the provided factory.

Parameters
typeThe type of the resource to add.
quantityThe quantity of the resource to add.
factoryThe resource factory used to create the resource if it does not exist.

Consume a specified quantity of a resource.

This method attempts to consume a specified quantity of the given resource type. If the resource is available and the consumption is successful, it returns true; otherwise, it returns false.
